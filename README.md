# React-Todo App

## Skills
Skills for this project:

- React ( React Hooks ) 
- Weather API
- Real Time 

This repository provides a code of the React-Todo application 
written using React Hook, React JSX and ES6.

## Running the example
```js
npm install
npm start
open http://localhost:3000
```
